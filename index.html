Att TeXa: en praktisk guide.

Note: This page describes a book about the LaTeX typesetting system written in Swedish. As such, the page is in swedish as well. If you are looking for an english-language guide to LaTeX, I can recommend both The Not So Short Introduction to LaTeX2e and Math Mode. If you have questions regarding LaTeX, check out the Stack Exchange TeX Site.

Att TeXa: en praktisk guide är en enkel introduktion till LaTeX för dig som läser på Chalmers Tekniska Högskola (eller dig som helt enkelt vill lära dig LaTeX). Boken är tänkt att ge en bra bas när det gäller enkel typsättning, vanliga fallgropar och typsättning av matematik, och baseras delvis på The Not So Short Introduction to LaTeX2e, Herbert Voß’ Math Mode och von Schultz Rekommendationer för LaTeX-dokument.

Om du inte redan visste det, så är LaTeX ett mycket kompetent typsättningssystem med rötterna i 80-talet, och som fortfarande används flitigt i till exempel akademiska kretsar. LaTeX gör det enkelt för dig som författare av böcker, rapporter, artiklar och liknande att skapa snygga, lättlästa dokument av hög kvalitet.

Boken innehåller en komplett introduktion till LaTeX, och efter att ha gått igenom bokens 6 kapitel bör du utan problem kunna typsätta dokument med LaTeX. Boken behandlar utöver LaTeX även matematikpaketet AmSLaTeX och bibliografisystemet biblatex.

Från början var boken tänkt att läsas av studenter vid programmen Teknisk Fysik och Teknisk Matematik vid Chalmers Tekniska Högskola, men alla som har ett intresse av att typsätta med LaTeX eller planerar att skriva långa dokument med många referenser eller mycket matematik kan (och bör) läsa boken.

Boken finns tillgänglig i PDF-format (i A4-storlek), samt som versionshanterad LaTeX-kod på Github (i både A4- och C5-storlek). Appendix A finns dessutom tillgängligt som en TeX-fil. Tanken är att A4-versionen ska läsas på skärm eller skrivas ut på en vanlig skrivare, medan C5-versionen kan tryckas i riktigt bokformat.

Den mer tekniskt kunnige kanske även vill kompilera boken själv. För detta krävs en fungerande kopia av XeLaTeX (som följer med TeXLive), ett antal paket från CTAN samt Pygments (med både Github-temat och Mathematica-lexern). För att checka ut koden från Github krävs dessutom Git. Man kan med fördel använda make för att kompilera hela boken i ett svep. Hittar du felaktigheter i koden (eller i brödtexten) får du gärna lämna in en buggrapport.
